# Guss Print Maker

Site vitrine e loja online para o **Guss Print Maker** — impressão 3D personalizada com temática geek (catálogo de produtos, carrinho, checkout, projeto sob medida, contato e área da conta com autenticação).

Desenvolvido com **React**, **TypeScript**, **Vite**, **Tailwind CSS** e integrações como **Supabase** e **React Router**.

## Como rodar o projeto

Na pasta do repositório, com [Node.js](https://nodejs.org/) instalado:

```bash
npm install
npm run dev
```

O Vite sobe o servidor de desenvolvimento (por padrão em `http://localhost:5173`). Abra esse endereço no navegador.
